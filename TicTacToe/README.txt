//README

Easy program to run, two files contain the ligic of this
class. One implements the JavaFX front side called 
TicTacToeGui. The file with the logic and the backside 
is in the file TicTacToeGame. Both reside in a package 
called game and they will try to look for the .jpg files 
one level above, at the same level as its package. Running
this program will trigger the GUI to start. Entry method is in GUI.
